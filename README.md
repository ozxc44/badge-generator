# GitHub Badge Generator

Create beautiful badges for your GitHub README without any build step.

## Quick Start

Just copy and paste! All badges are SVG-based and work everywhere.

## Status Badges

```
[![Status](https://img.shields.io/badge/status-active-success)]()
[![Status](https://img.shields.io/badge/status-maintenance-warning)]()

```


Replace `your_color` with any CSS color name or hex code.
